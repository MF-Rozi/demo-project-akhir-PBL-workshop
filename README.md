# Asia Heritage Pekanbaru - Tourism Information System

A Laravel-based tourism information system for Asia Heritage Pekanbaru featuring zones, attractions, visitor reviews, and admin management.

## Features

### Public Features
- **Landing Page**: Browse tourism zones and attractions with beautiful Bootstrap 5 UI
- **Attraction Details**: View detailed information, images, and visitor reviews
- **Review System**: Submit 5-star ratings and text reviews without login
- **Responsive Design**: Optimized for all devices

### Admin Features  
- **Dashboard**: Overview statistics and recent reviews
- **Zones Management**: Create, edit, delete tourism zones with image uploads
- **Attractions Management**: Manage attractions with zone assignment and images
- **Review Moderation**: Approve/reject/delete visitor reviews
